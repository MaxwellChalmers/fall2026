title: Verification Guide — Career Modules
---

## What to check

**Frontmatter**
- `title`, `type`, `draft`, `status`, `priority` all set
- `draft` is `0` if ready to publish
- `scheduled_day` is set if tied to a specific class meeting

**Required sections**
- **Overview** — brief explanation of what this module covers and what students will get out of it
- **Before Class** — any prep work students need to do; instructions must be complete and links must work

**Link checks — do these first**
- Open every external link in the module and confirm it loads
- PathwayU links, Google Doc links, and external career resources go stale; verify each one
- If a link requires a login, confirm UNCA students have access

**Content quality**
- Instructions are written in plain language — a first-year student should be able to follow them
- No placeholder text, draft notes, or bracketed TODOs visible on the page
- Deliverables (if any) are clearly described

## How to mark verified

```
status: verified
status_reviewer: [your name]
status_date: [YYYY-MM-DD]
status_notes: [optional — especially note any broken links or access issues]
```
